# Viola Community

Русскоязычный фан-портал ACF Fiorentina: новости, трансферы, календарь матчей,
страницы материалов и простая админка для ручного управления контентом.

## Что есть в проекте

- Главная страница с hero-блоком, новостями, матчами, fan-zone и footer.
- Раздел новостей и трансферов с отдельными страницами материалов.
- Календарь матчей и match-center.
- Админка по адресу `/admin`.
- Редактирование новостей, трансферов, матчей, настроек и главной страницы.
- Загрузка изображений в админке с автоматическим отображением в нужных блоках.
- Экспорт и импорт JSON-данных для переноса контента.

Проект рассчитан на работу без сложного backend/VPS: данные админки хранятся в
браузере через `localStorage`. Перед переносом на другой компьютер или браузер
нужно сделать экспорт JSON из админки и затем импортировать его обратно.

## Требования

- Node.js `>=22.13.0`

## Запуск локально

```bash
npm install
npm run dev
```

На Windows можно запускать так:

```bash
npm.cmd run dev
```

После запуска:

- сайт: `http://localhost:3000`
- админка: `http://localhost:3000/admin`

## Проверка

```bash
npm test
```

Команда собирает проект и проверяет базовый HTML-рендер.

## Полезные команды

```bash
npm run build
npm run lint
npm run db:generate
```

## Важные заметки

- `.env*` файлы не коммитятся.
- Временный файл `telegram-acf.html` не коммитится.
- Папка `build/` нужна проекту: `vite.config.ts` импортирует локальный Sites
  plugin из `build/sites-vite-plugin.ts`.
- Если контент менялся через админку, перед деплоем или переносом сделайте
  экспорт JSON в админке.
